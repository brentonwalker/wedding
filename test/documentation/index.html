HTML Structure

This wedding page template is a responsive layout which caters to both desktop and mobile devices, it is built with the Foundation 4 Framework.

For detailed documentation on how Foundation works I recommend you check out this link - http://foundation.zurb.com/docs/v/4.3.2/.

Eternity can be easily customized to suit your needs. Be sure to use the psd templates provided to match the exact dimensions of the structure.
In addition, all markup has been clearly commented to assist in indefying where the correspoding sections begin and end as required.

If you do have any difficulties feel free to contact me directly and I'll be happy to help you out

CSS Files and Structure

Eternity wedding template page uses only 7 CSS files. A brief explanation of each is outlined below:

  1. foundation.min.css - contains all core styles of the template - courtesy of the Foundation 4 framework (there should be no need for you to use this file);
  2. normalize.min.css - makes all browsers render elements consistently. For further information click here
  3. font-awesome.min.css - provides all the icon fonts for the template.
  4. style.css - contains all of the overrides and custom styles for the overall template. This is the file your should use when creating new selectors or modifying existing ones;
  5. media-queries.css - contains all of the required media queries for the template responsive functionality. I've separated this file to allow for easier customization of any device that may be required without affecting the mobile first implementation;
  6. flexslider.css - contains all of the rulesets for the flexslider carousel.
  7. ie8.css - contains all of the rulesets for the magical world of Internet Explorer 8.

Note: If you would like more information on what classes are contained within these, I would recommend you check out this link - http://foundation.zurb.com/docs/v/4.3.2/.

JavaScript

This template imports 9 Javascript files. A brief explanation of each is outlined below:

  1. jquery.min.js - The library that powers and provides support for all jQuery plugins.
  2. custom.js - a collection of custom functions allowing for the specific behaviours of the template such as; smooth scrolling, Google Map API integration and more. Use this file if you need to add new custom functions or edit existing ones.
  3. custom.modernizr.js - toolkit that allows web developers to use new CSS3 and HTML5 features.
  4. foundation.min.js - used for to initiate all Foundation 4 core plugin functionality. The minified version is used for the template.
  5. foundation.abide.js - used for the RSVP form client side validation.
  6. jquery.scrollTo-1.4.3-min.js - used for smooth scrolling
  7. jquery.nav.min.js - plugin for the navigation on one-page sites which also adds smooth scrolling effects.
  8. jquery.flexslider.js - powers the fully responsive slider carousel
  9. jquery.backstretch.min.js - a simple jQuery plugin that allows you to add a dynamically-resized, slideshow-capable background image

I would recommend not making any changes to these files unless you have prior knowledge of JS and JS libraries.

PHP

This template includes one PHP file which allows for the RSVP form to send an email to your specified address.

You can find this file under the path below.

\main-download\html\mailer.php

The main configuration options are found within this file (mailer.php) and will need to be updated to the options you would like. All code is clearly commented for you to easily determine what details should be updated.

One last thing to mention is the form uses an AJAX callback which means no page refresh will occur on form submission. This is a better way to submit forms as the end user will be able to continue on the page with no changes to their view.

If you need more information on how to configure the form feel free to contact me.
